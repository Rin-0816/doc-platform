# ICT 教材プラグイン仕様書

## 1. 目的

ICT 学習資料向けの追加機能を定義する。

## 2. 責任範囲

- コアは `ict_learning` の field 名や保存先を知らず、host API 経由で runtime を呼び出す
- ICT 教材 runtime は `plugin_data.ict_learning` の読込、保存、復元を担う
- ICT 教材 runtime は専用 UI、validation、検索拡張、追加コンテンツ定義を登録する
- 無効化時は作成ツールとビューアーの ICT panel、検索拡張、validation を停止し、専用テーブルとリビジョンスナップショットは保持する
- 再有効化時は保持済みデータを再利用する

## 3. 追加メタデータ

- 学習目標
- 前提知識
- 難易度
- 想定学習時間
- 必要機材
- 必要ソフトウェア
- 対応 OS
- 対応バージョン
- 最終確認日

## 4. 追加コンテンツ

- 演習
- 確認テスト
- トラブルシュート

## 5. UI 拡張

- 作成ツールの panel host に教材情報入力欄を登録する
- ビューアーの panel host に教材概要パネルを登録する
- 検索条件に難易度、必要機材、対応バージョンを追加する
- コアの HTML、CSS、JavaScript へ ICT 固有 markup や field 名を直書きしない

## 6. 検証

- 想定時間は正の値
- 最終確認日は日付形式
- 必要機材、必要ソフトウェアは複数登録可

## 7. テスト観点

- runtime による `plugin_data.ict_learning` の読込、保存、復元
- メタデータ保存
- 作成ツールの panel host 表示
- ビューアーの panel host 表示
- 検索反映
- 入力検証
- 無効化中は ICT UI と検証が止まり、データが保持されること
- 再有効化後に保持済みデータが再表示されること
