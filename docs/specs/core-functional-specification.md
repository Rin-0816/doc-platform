# コア機能仕様書

## 1. 目的

メインプログラムが標準提供する機能を定義する。

## 2. コア機能

- 認証と権限
- 文書管理
- セクション管理
- 添付管理
- タグ、カテゴリ、レッスン
- 用語管理
- コメント
- 履歴と差分
- 検索
- プラグイン管理
- 互換性チェック

## 3. 文書管理

- Markdown 文書の作成、編集、削除
- 画像添付
- Mermaid 記法の保持
- 現在版と履歴の管理

## 4. 整理機能

- タグ
- カテゴリ
- レッスン
- 用語
- 最近更新

## 5. 検索

- 文書本文
- タイトル
- タグ
- カテゴリ
- 用語

## 6. 拡張連携

- 文書保存時にプラグイン検証を呼ぶ
- 文書表示時にプラグイン表示パネルを差し込む
- 検索対象にプラグイン拡張項目を加える

## 7. テスト観点

- CRUD
- 権限制御
- 検索
- 添付
- 履歴連携
- プラグイン連携
